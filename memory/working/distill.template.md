# End-of-session distillation checklist

Run through this before context is lost. Not every session produces a
memory entry — most don't.

1. Did anything happen that's worth recording as a past event
   (success, failure, surprising outcome)? → new `episodic/` entry.
2. Did I learn a durable fact/preference that will still be true next
   session? → new `semantic/` entry.
3. Did a workflow prove itself reusable across more than one occasion?
   → new/updated `procedural/` entry.
4. Is there something to follow up on later? → new `prospective/` entry
   with a `due:` date.
5. If none of the above — do nothing. Not distilling is the correct
   outcome for most sessions.
